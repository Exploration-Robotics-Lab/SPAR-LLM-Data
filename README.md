# SPAR-LLM-Data
AUV fault-recovery dataset generated by SPAR-LLM at JHU IAA Exploration Robotics Lab. Recovery missions planned by NVIDIA NIM (nemotron-3-super-120b-a12b). Post-run telemetry and figures analyzed by NVIDIA NIM vision (llama-3.2-90b-vision-instruct). Nemotron fine-tuning for Jetson AGX Orin.
